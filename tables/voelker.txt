Spezialregel für Menschen: "Fokuswahl"
Ein Mensch erhält zu Beginn +1 auf alle sechs Attribute. Danach muss der Spieler eine der folgenden drei Optionen wählen, die eine Stärke mit einer Schwäche koppelt:
1) Fokus auf Stärke: Der Malus von -2 wird auf Intelligenz (INT) angewendet. (Finales Ergebnis für INT: -1)
2) Fokus auf Geschicklichkeit: Der Malus von -2 wird auf Weisheit (WIS) angewendet. (Finales Ergebnis für WIS: -1)
2) Fokus auf Konstitution: Der Malus von -2 wird auf Charisma (CHA) angewendet. (Finales Ergebnis für CHA: -1)
